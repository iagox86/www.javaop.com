JavaOp is a cross-platform, flexible, plugin-oriented Battle.net Binary Bot.

It is currently being developed mainly by Ron, based originally on the code for JBBot. Currently on the second major revision, using a bot based on Java for Battle.net is unprecidented, and graciously accepted by many members of the Battle.net community. It is being actively developed and tested, and all code from it is considered Public Domain. Feel free to Email me or post on my forum if you wish to contact me, or have a look at other pages on this site for more information.

August 24, 2007

I guess I should start by saying that I'm still alive, and I still check my email every day (although the @valhallalegends.com address is dead, use the one above). The forum still exists, too, and JavaOp2 should still work on Battle.net.
